Файловая система (для дальнейшей автоматизации) про ведению учета денежных средств.
Дожна заполняться каждый день в ручную (в дальнейшем автоматически).
На текущий год создается файловая система с Excel файлами на каждый квартал.
Исходные данные по счетам будут браться из текстового документа (добавить).
Для корректной работы программы необходимо закрывать все Excel файлы перед запуском скрипта.
